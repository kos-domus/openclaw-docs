## Operator-relevant highlights

- Skill Workshop becomes a guarded lifecycle: `skill_research` can apply, reject, quarantine, revise, hash, scan, and roll back pending `PROPOSAL.md` skill drafts with approved support files under standard skill folders.
- External plugin packaging keeps moving: Tokenjuice and GitHub Copilot are official install-on-demand plugins with npm and ClawHub metadata, while SecretRef provider integration manifests and shared LLM core packages improve plugin/provider reuse.
- Workboard adds orchestration primitives and agent coordination tools for multi-agent planning and run tracking, which is directly relevant to fleet operations.
- Cron compatibility and resilience improve: SQLite migrations stay compatible with legacy run-log tables, archived job stores, diagnostic cron names, and legacy one-shot delete-after-run behavior; recurring jobs retry transient model rate limits before waiting for the next scheduled slot.
- Channel and provider reliability is the main hardening line: Telegram, Discord, WhatsApp, Signal, Feishu, Google Chat, Microsoft Teams, QQBot, Nostr, Zalo, and Nextcloud-style timers/retries are capped; local service, model, usage, queue, media, TTS, music, workflow polling, and provider OAuth calls are bounded.
- Agent/Codex cleanup continues: live locks survive cleanup, interrupted CLI transcripts recover, orphan tool state is cleared, app-server idle timers are capped, and media completion delivery remains retryable.
