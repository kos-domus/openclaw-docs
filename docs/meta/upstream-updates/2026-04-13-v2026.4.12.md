# OpenClaw 2026.4.12

A broad quality release focused on plugin loading, memory and dreaming reliability, local-model options, and smoother setup paths.

## Highlights

- New optional Active Memory plugin with a dedicated memory sub-agent before the main reply
- Experimental local MLX speech provider for Talk Mode on macOS
- New `openclaw exec-policy` command family for syncing local exec policy and approvals
- New `commands.list` RPC for runtime command discovery
- Provider request controls for trusted self-hosted OpenAI-compatible endpoints
- Continued Dreaming, UI, and plugin stability work

## Relevance for this docs repo

The new Active Memory and exec-policy surfaces likely deserve future reference coverage if they become part of local operating practice.
