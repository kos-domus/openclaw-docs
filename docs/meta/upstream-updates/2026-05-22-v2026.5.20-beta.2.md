# OpenClaw 2026.5.20-beta.2

This is basically the beta checkpoint immediately preceding the `2026.5.20` stable cut, so the value here is mostly confirming what just landed on stable.

## Highlights

- Includes the bundled Policy plugin work, xAI device-code OAuth, and the Discord voice/realtime follow-user flow.
- Keeps the bounded bootstrap-context-file behavior for realtime voice sessions.
- Confirms the same Codex/runtime and agent-config direction that shipped in stable.

## Breaking or operator-impactful changes

- Same skill-wrapper auto-allow removal as the stable cut: no more compatibility crutch for `cat SKILL.md && ...` wrappers.
- More agent-scoped config surface means more power, but also more room for silent config drift.

## Fixes most worth caring about

- WhatsApp/Baileys and outbound-drain cleanup.
- Cron correctness and result classification fixes.
- Better subagent completion recovery and requester handoff.
- Better visibility into model pinning and session/runtime state.
