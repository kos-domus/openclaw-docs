# OpenClaw 2026.5.3-1

Small npm hotfix on top of `2026.5.3`.

## Highlights

- Relaxed the plugin install security scanner so official bundled plugin packages are not falsely blocked when benign `process.env` access or normal API sends exist elsewhere in the same compiled bundle.

## Breaking or operator-impactful changes

- No new breaking change called out, but this matters if official plugin installs/updates were being rejected after moving to `2026.5.3`.

## Relevance for this docs repo

This hotfix mainly reduces false-positive friction in official plugin install/update flows. If we upgrade past `2026.5.2`, it is safer to land on `2026.5.3-1` rather than plain `2026.5.3`.
