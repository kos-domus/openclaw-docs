- Stable advanced from `2026.6.6` to `2026.6.8` on both GitHub latest release and npm `latest`.

- Agent and Gateway recovery improved around account-scoped DM sends, generated media completions, auto-reply message-tool final replies, reset archive fallback reads, restart shutdown aborts, yielded subagent pauses, heartbeat dedupe, session identity prompts, and unknown OpenAI agent selector rejection.

- Web search defaults are safer: key-free providers such as Parallel Free, DuckDuckGo, Ollama, and Codex Hosted Search remain explicit opt-ins rather than automatic fallback choices.
- UI, mobile, memory, and state handling improved across collapsed workspace files, WebChat backscroll, session picker interactivity, stale iOS Gateway reconnects, oversized OpenAI embedding batches, QMD transient search, SQLite on NFS volumes, and full-reindex rollback/cache recovery.
