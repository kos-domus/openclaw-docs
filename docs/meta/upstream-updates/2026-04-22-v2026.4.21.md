# OpenClaw 2026.4.21

A small follow-up release focused on image-generation defaults, plugin repair via doctor, command-owner enforcement, and a handful of targeted channel/browser/install fixes.

## Highlights

- Bundled image generation now defaults to `gpt-image-2`, with updated 2K/4K metadata and docs hints
- `openclaw doctor` can repair bundled plugin runtime dependencies from doctor paths in packaged installs
- Owner-only commands now require real owner identity instead of permissive wildcard fallbacks
- Targeted fixes landed for Slack thread aliasing, browser accessibility refs, and npm override hygiene

## Relevance for this docs repo

This is mostly a maintenance release, but the doctor repair path and owner-command hardening are worth tracking because they affect troubleshooting, security posture, and operator expectations.
