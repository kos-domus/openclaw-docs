# OpenClaw 2026.5.12-beta.8

Last beta checkpoint before stable. The strongest signals are optional-provider/plugin externalization, Telegram ingress durability, and cleaner auth/config behavior.

## Highlights

- Bedrock, Slack, Anthropic Vertex, and sandbox-adjacent surfaces keep moving toward optional plugin installs instead of core bloat.
- Telegram durability improved again, including ingress isolation and better handling of mention-gated media.
- Config/auth behavior got a useful cleanup pass, especially around env-var credential inference and onboarding flag propagation.

## Breaking or operator-impactful changes

- Good operator signal, but still a broad beta line with runtime, auth, Telegram, plugin, and gateway touches.

## Fixes most worth caring about

- Telegram polling now survives main event-loop stalls better thanks to isolated ingress/spooling.
- Config/auth stopped inferring credentials from generic uppercase env-like strings, which is a nice safety hardening.
- `sessions_send` / agent bootstrap and rich reply delivery paths kept getting cleaned up before stable.
